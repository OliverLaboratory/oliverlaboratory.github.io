How it works

Standard proof of work burns energy on hashes that mean nothing outside the chain. crick keeps Bitcoin's security model but adds one rule that makes solving real problems the rational mining strategy.

01 · TWO WAYS TO MINE

Solve, or hash

Every block can be mined classically at the base difficulty d_b, or — if it carries a strictly better solution to the network's puzzle — at a reduced difficulty d_r < d_b. Improving the solution is cheaper than out-hashing the network.

02 · DIPS v2 RETARGETING

Independent difficulties

Each difficulty retargets from its own block type: d_b holds classical production steady, d_r holds solution production steady. When a full window passes with no solution found, d_r drops 4× — the harder the problem gets, the deeper the discount for solving it.

03 · USEFUL OUTPUT

Science on the ledger

The chain becomes a public, append-only record of ever-improving biological solutions. Ships with two seed problems: protein docking (lowest-energy ligand poses, scored by a fixed integer energy function) and maximum common subgraph of protein contact graphs (shared structural motifs). Instances are drawn from billions × billions of structures — so vast that precomputing them is the science.

Install

crick is a single Python package with one small dependency. Python 3.9+ required.

# from source
$ git clone <repo-url> crick && cd crick
$ pip install .

# or, isolated install with pipx
$ pipx install ./crick

That's it — the crick command is now on your PATH.

Quickstart

Create a wallet and genesis block, then start mining — solo or against the network.

$ crick init                  # wallet + genesis block in ~/.crick
$ crick mine                  # mine solo: solver + hasher in one loop
$ crick node --mine \
        --peer http://seed.example:9911   # join a network and mine
$ crick status                # chain height, difficulties, balance
$ crick solution              # the best on-chain solution so far
$ crick send --to crk1… --amount 5   # spend your rewards
CommandWhat it does
crick initCreate a wallet and the genesis block (stores the puzzle instance)
crick mineMine solo — alternates puzzle solving with nonce search
crick nodeRun a full node: HTTP API, peer gossip, optional mining
crick statusHeight, total work, d_b, d_r, best clique, balance
crick solutionShow and verify the best solution recorded on-chain
crick sendSign and broadcast a transaction via a running node

The papers

crick implements the protocol proposed and refined in two papers. Where they differ, the 2019 difficulty-scaling scheme (DIPS) is authoritative.

Proposal for a fully decentralized blockchain and proof-of-work algorithm for solving NP-complete problems
Oliver, Ricottone, Philippopoulos · 2017 · arXiv:1708.09419
Difficulty Scaling in Proof of Work for Decentralized Problem Solving (DIPS)
Philippopoulos, Ricottone, Oliver · 2019 · arXiv:1911.00435
